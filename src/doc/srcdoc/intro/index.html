Introduction to OpenKore development

Last update: June 15, 2004

This guide is still under construction. Feel free to contribute.

What this guide is

This guide is written to help new developers getting started with OpenKore development.

What this guide is not

  • This guide was created to help people who already have experience with programming with getting started on Kore development.
  • This guide is not meant for people who have no programming experience at all.
You should have at least the following knowledge:
  • Understanding of the difference between a compiled and interpreted language.
  • Understanding of network programming.
  • Some experience with DOS or command lines (not required but it will help you a lot).
