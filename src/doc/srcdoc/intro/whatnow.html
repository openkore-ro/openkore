So, what now?

Uhm yeah great, now you know how to use Perl, but what now? That might be what you're thinking right now. You could:
  • Try to fix a bug if you've found one. This is also a good way to get more familiar with the code base.
  • Implement cool features that you might have in mind.

The TODO list

Are the above points too abstract? Need something specific? I've compiled a TODO list exactly for this purpose: http://openkore.sourceforge.net/forum/viewtopic.php?t=1309
Some items aren't even coding tasks or technical tasks.
