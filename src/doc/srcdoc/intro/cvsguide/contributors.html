<< OpenKore CVS Guide

Important note

The SourceForge anonymous CVS servers are known to be buggy. They tend to be down from time to time. If you're unable to access the CVS server right away, wait a while and try again.

1. Download (checkout) source code from CVS

Go to the folder in which you want to put the source code. Rightclick on an empty spot in the folder, and click on "CVS Checkout...".
A dialog will appear. Fill in this for the "CVSROOT" field:
:pserver:anonymous@cvs.sourceforge.net:/cvsroot/openkore
For the "Module" field, enter a module name (see the previous page for a list of modules).
When you're done, click on OK. The code will now appear in a subfolder with the same name of the module. You can now modify the source code as you wish.

2. Modify the source code

This is up to you. ;) However, you should read the Developers Notes first if you haven't done so. If you've downloaded the openkore module, read DevelopersNotes.txt.

Tip: you may have already noticed by now that all files inside the module folder are green. Files that you have changed will automatically turn red.

3. Tell CVS to merge your changes with the latest changes in CVS

Your changes will not be in the CVS repository until you commit them.

By the time you're done modifying the code, another developer may have already uploaded his changes to CVS. This is why you should always do a CVS update: merge your changes with possible new changes in CVS.

Rightclick on an empty spot in the module's folder and choose "CVS Update".
Note: sometimes CVS may report conflicts during merging. For more information, read "How to resolve merging conflicts".

This command is also used to download the latest source code from CVS, not just to merge your changes.

4. Create a patch and submit it to the forums

A patch is a text file that contains the changes you've made. Select all the files that you want to make a patch for. Rightclick and choose "CVS Diff..." or "CVS->Make patch". TortoiseCVS will ask you where to save the patch file. Make sure you save it with the .patch or .diff extension.

After having saved the patch file, go to the forums, section Addons, Mods & New Code. Make a new topic. There are two ways to post your source code:

  1. Copy & paste your code from Notepad to the web browser. Put the source code between [CODE] UBB tags (recommended way).
  2. Add your patch file as attachment.
